display é a forma como o navegador organiza e apresenta cada elemento na tela.

block → O elemento ocupa toda a largura disponível e sempre começa em uma nova linha.
inline → O elemento fica na mesma linha que os outros e só ocupa o espaço do seu conteúdo.
inline-block → Mistura dos dois: fica na mesma linha, mas permite definir largura e altura.
flex→ Transforma o elemento em flex container, permitindo organizar os filhos com Flexbox.
grid→ Transforma o elemento em grid container, permitindo organizar os filhos em linhas e colunas.
none→ Faz o elemento sumir da tela (não ocupa espaço).

Observação: O valor padrão do display depende da tag HTML:
Elementos de bloco (block elements) → padrão é display: block; Ex: div, p, h1, section, header.
Elementos em linha (inline elements) → padrão é display: inline; Ex: span, a, strong, em.

Flexbox

Propriedades para o pai (flex container)

flex-direction controla se os elementos ficam lado a lado ou um embaixo do outro, e também a ordem em que aparecem.
row (padrão)→ os itens ficam em linha, da esquerda para a direita.
row-reverse→ os itens ficam em linha, mas da direita para a esquerda.
column→ os itens ficam em coluna, de cima para baixo.
column-reverse→ os itens ficam em coluna, mas de baixo para cima.

Flex-direction na prática

justify-content controla a distribuição horizontal (ou vertical, dependendo do flex-direction) dos elementos dentro de um container flexível.
flex-start (padrão)→ itens alinhados no início.
flex-enditens alinhados no final.
center → itens centralizados.
space-between→ espaço igual entre os itens, sem espaço nas bordas.
space-around→ espaço igual ao redor de cada item.
space-evenly→ espaço totalmente igual entre os itens e nas bordas.

Justify-content na prática

align-items define como os elementos ficam alinhados verticalmente (quando flex-direction: row) ou horizontalmente (quando flex-direction: column).
stretch (padrão) → os itens esticam para ocupar toda a altura (ou largura, se flex-direction: column).
flex-start → alinha os itens ao início do eixo cruzado.
flex-end→ alinha os itens ao final do eixo cruzado.
center→ centraliza os itens no eixo cruzado.
baseline → alinha os itens pela linha de base do texto.

Align-items na prática

align-content align-content alinha grupos de linhas dentro do container flex. ⚠️ Ele só funciona se houver mais de uma linha de itens (com flex-wrap: wrap;).
stretch (padrão) → as linhas se esticam para ocupar todo o espaço disponível.
flex-start → todas as linhas ficam juntas no início.
flex-end → todas as linhas ficam juntas no final.
center → todas as linhas ficam centralizadas.
space-between→ espaço igual entre as linhas, sem espaço nas bordas.
space-around→ espaço igual ao redor de cada linha.
space-evenly→ espaço totalmente igual entre linhas e bordas.

Propriedades para crianças (itens flexíveis)

order É uma propriedade do Flexbox que permite alterar a ordem de exibição de um item filho, sem precisar mudar a ordem no HTML.
Por padrão, os itens flexíveis são dispostos na ordem de origem. No entanto, a order propriedade controla a ordem em que eles aparecem no contêiner flexível.

Order

flex-grow: controla quanto cada item aumenta de tamanho em relação aos outros dentro de um container flex.
0 (padrão) → o item não cresce.
1 → o item cresce proporcionalmente ao espaço disponível.
2,3... → cresce mais rápido, proporcional ao valor em relação aos outros itens.

flex-grow

flex-shrink controla como os itens diminuem de tamanho quando o container não tem espaço suficiente.
0 → o item não encolhe (permanece do tamanho original).
1 (padrão) → o item pode encolher proporcionalmente aos outros itens.
Valores maioreso item encolhe mais rápido que os outros.

flex-shrink

flex-basis define o tamanho inicial de um item dentro de um container flex, servindo como base para os ajustes de crescimento (flex-grow) ou encolhimento (flex-shrink).
Pode ser um valor em pixels, porcentagem, ou auto:
auto (padrão) → usa o tamanho natural do item.
100px, 30%→ define um tamanho fixo inicial antes do flex-grow ou flex-shrink entrarem em ação.
Funciona como largura ou altura inicial, dependendo do flex-direction:
row → flex-basis define largura inicial.
column → flex-basis define altura inicial.

flex-basis

flex é uma forma prática de definir crescimento, encolhimento e tamanho inicial de um item flex em apenas uma linha de código.
